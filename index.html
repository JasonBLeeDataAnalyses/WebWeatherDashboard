Summary: Latitude vs. X

The main objective of this assignment was to determine the effect on weather as the latitudes approach the equator. Data was pulled from OpenWeatherMap API using a random sample of over 500 cities.


Using Python CitiPy library and OpenWeatherMapAPI, I got the weather for the cities (using the nearest lat-lng) and assembled a dataset of weather information for over 500 cities.


The data and plots are displayed on this website along with descriptions and possible explanations of any trends or correlations.
